Phonetic equivalence
First consonant: There are many cases where the first consonant is systematically replaced by another in Quang Binh dialect. For example
- nh → d, gi (Standard → Quang Binh Dialect)
nhà → già
nhờ cậy → giờ cậy
nhiều ít → diều ít
nhòm ngó → dòm ngó
- nh → l (Standard → Quang Binh Dialect)
hoa nhài → hoa lài
nhạt → lạt
nhanh → lanh
- s → th (Standard → Quang Binh Dialect)
ngôi sao → ngôi thao
củ sắn → củ thắn
- x → th (Standard → Quang Binh Dialect)
xếp giấy → thếp giấy
cái xuổng → cái thuổng
- v → b (Standard → Quang Binh Dialect)
vui vẻ → bui bẻ
vá → bá
vo → bo
véo → bẹo
vừa → bừa
- v → ph (Standard → Quang Binh Dialect)
vỗ tay → phổ tay
ăn vụng → ăn phúng
- b → ph (Standard → Quang Binh Dialect)
(nói) bịa → (nói) phịa
bỏng → phỏng
- d → đ (Standard → Quang Binh Dialect)
con dam → con đam
dưới → đưới
- d → th (Standard → Quang Binh Dialect)
dỗ (dỗ dành) → thổ
dấu → thu
- d → r (Standard → Quang Binh Dialect)
dở hơi → rở hơi
- g → kh/c (Standard → Quang Binh Dialect)
gãi → khải
gỡ → khở
gõ → khỏ
- gi → tr (Standard → Quang Binh Dialect)
giun → trùn
ở giữa → ở trữa
già giặn → tra trắn
đầu gối → trôốc cún
- gi → ch  (Standard → Quang Binh Dialect)
giường → chờng
giờ → chừ
- c → n (Standard → Quang Binh Dialect)
cạo → nạo
